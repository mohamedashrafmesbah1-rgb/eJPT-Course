# 🛡️ eJPT (eLearnSecurity Junior Penetration Tester)

## 📌 Overview
This repository documents my learning journey in the eJPT certification, which is a hands-on penetration testing course focused on building practical offensive cybersecurity skills.

---

## 🎯 Course Objectives
- Understand penetration testing methodology  
- Learn reconnaissance and information gathering techniques  
- Perform network scanning and enumeration  
- Identify and exploit vulnerabilities in lab environments  
- Build foundational ethical hacking skills  

---

## 📚 Topics Covered
- Footprinting & Reconnaissance  
- Network Scanning (Nmap)  
- Enumeration Techniques  
- Vulnerability Identification  
- Basic Exploitation Techniques  
- Introduction to Web Application Attacks  
- Post-Exploitation Basics  

---

## 🛠️ Tools Used
- Nmap  
- Netcat  
- Wireshark  
- Metasploit Framework (basic usage)  
- Linux terminal tools  
- Virtual Lab environments  

---

## 💡 Skills Gained
- Practical penetration testing workflow  
- Network and system reconnaissance  
- Vulnerability analysis  
- Hands-on ethical hacking experience  
- Understanding attacker mindset

## 📌 Purpose
This course helped me build a strong foundation in offensive cybersecurity and prepared me for more advanced penetration testing certifications and real-world security assessments.
